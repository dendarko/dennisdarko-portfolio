## Checklist

- Define one measurable user workflow and success metric.
- Separate prompt logic, retrieval, and orchestration concerns.
- Add request/response schema validation.
- Add rate limiting, auth, and secrets management.
- Implement tracing, logs, and token/cost metrics.
- Create smoke tests and regression evaluation set.
- Add fallback behavior for provider outages/timeouts.
- Gate deployment with CI checks.
- Version prompts, models, and datasets.
- Document rollback procedure.
- Review privacy and retention policies.
- Pilot with a small user cohort before full rollout.
